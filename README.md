# HR Analytics Dashboard – Employee Retention Analysis

# Overview
This project is an interactive HR Analytics Dashboard developed in Microsoft Excel to analyze employee attrition and retention patterns. The dashboard helps HR teams identify key factors affecting employee turnover through data visualization and KPI analysis.

---

# Objectives
- Analyze employee attrition trends
- Identify high-risk departments and job roles
- Monitor workforce demographics
- Evaluate salary and experience impact on attrition
- Support data-driven HR decision-making

---

# Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Conditional Formatting
- Slicers
- KPI Cards
- Data Visualization

---

# Dashboard Features
- Total Employees KPI
- Attrition Rate Analysis
- Department-wise Attrition
- Gender-wise Attrition
- Salary Band Analysis
- Age Group Analysis
- Job Satisfaction Analysis
- Interactive Filters & Slicers

---

# Key Insights
- Overall attrition rate is 16%
- Employees aged 26–35 show highest attrition
- R&D department has maximum attrition
- Low salary employees leave more frequently
- Sales Executive and Lab Technician roles show high attrition
